# README Generation

Generate or update README.md for the current project:

1. Analyze project structure (package.json, composer.json, main files)
2. Identify key features and dependencies
3. Include installation instructions
4. Add usage examples with code snippets
5. Document configuration options
6. Follow WordPress/npm conventions as appropriate
7. Output in Markdown format with proper headings
